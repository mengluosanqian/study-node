# 手写
1. 实现⼀个函数a,使其奇数次调⽤时返回1,偶数次调⽤时返回2,不能使用全局变量
```js
function A() {
    // 记录调用的次数
    let count = 0
    return function () {
      count++
      return count % 2 === 0 ? 2 : 1
    }
  }
  let aa = A()
  console.log(aa());
  console.log(aa());
  console.log(aa());
```
